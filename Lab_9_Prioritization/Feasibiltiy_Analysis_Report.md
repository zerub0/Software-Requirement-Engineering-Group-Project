# Feasibility Analysis Report - Bus Ticketing System

---

## Objective

To evaluate the feasibility of implementing requirements for the Bus Ticketing System given the project's constraints.

---

## What is Feasibility Analysis?

Feasibility analysis evaluates whether a requirement can be implemented given the project's constraints including technical, economic, legal, operational, and schedule factors.

---

## Feasibility Analysis Summary

| **Requirement** | **Feasibility Status** | **Justification** |
|-----------------|------------------------|-------------------|
| FR-05: Make Payments | Feasible with Constraints | Requires integration with third-party payment gateway and security compliance (PCI-DSS) |
| NFR-02: 99.9% Uptime | Feasible with Constraints | Requires investment in reliable hosting infrastructure, load balancing, and backup systems |
| All Other Requirements | Feasible | Can be implemented with existing technology and available resources |

---

## Assumptions and Constraints

| **Type** | **Description** |
|----------|-----------------|
| Assumption | Users have access to smartphones or computers with internet |
| Assumption | Payment gateway provider is reliable and secure |
| Assumption | Bus operators are computer literate |
| Constraint | Budget is limited; prioritize essential features |
| Constraint | Timeframe is 6 months for the first release |
| Constraint | Must comply with transport authority regulations |

---

## Conclusion

| **Aspect** | **Conclusion** |
|------------|----------------|
| **Technical Feasibility** | All requirements are technically feasible with current technology |
| **Economic Feasibility** | Most requirements are economically feasible; payment integration requires additional budget |
| **Operational Feasibility** | All requirements are operationally feasible |
| **Schedule Feasibility** | Requirements can be delivered within 6 months with proper prioritization |
| **Overall Assessment** | The project is feasible with some constraints that need to be managed |
