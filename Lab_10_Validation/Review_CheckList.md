# Review Checklist - Requirements Validation

## Bus Ticketing System

---

| **Checklist Item** | **Description** |
|--------------------|-----------------|
| Clarity | Is the requirement easy to understand? |
| Completeness | Is the requirement fully specified? |
| Consistency | Does it conflict with other requirements? |
| Testability | Can the requirement be tested? |
| Measurability | Is it measurable (especially for NFRs)? |
| Uniqueness | Does it have a unique ID? |
| Feasibility | Is it technically feasible? |
| Traceability | Can it be traced to a stakeholder need? |
